Getting Started

Welcome to the Zoning Compliance Checker API documentation. This guide will help you integrate our powerful zoning compliance analysis into your applications.

Base URL

http://localhost:5000/api

Quick Start

  1. Obtain an API key (Enterprise accounts only)
  2. Make a request to the compliance analysis endpoint
  3. Parse the JSON response for compliance results

API Reference

Response Format

All API responses are returned in JSON format with a consistent structure:

{
  "success": true,
  "data": { ... },
  "message": "Operation completed successfully"
}

Authentication

Enterprise API endpoints require authentication via API key header:

X-API-Key: your_api_key_here
Note: Government authentication endpoints use session-based authentication with JWT tokens. See the login endpoint for details.

Endpoints

GET /api/zones

Get all available zone types.

Response:
{
  "success": true,
  "zones": ["R-1", "R-2", "R-3", "C-1", "C-2", ...],
  "count": 9
}
POST /api/geocode

Convert an address to geographic coordinates.

Request Body:
{
  "address": "1600 Pennsylvania Avenue, Washington, DC"
}
Response:
{
  "success": true,
  "coordinates": {
    "latitude": 38.8977,
    "longitude": -77.0365,
    "accuracy": 0.9
  },
  "address": "1600 Pennsylvania Avenue, Washington, DC"
}
POST /api/compliance/analyze

Perform compliance analysis for a property. Requires Auth

Request Body:
{
  "zone_type": "R-1",
  "coordinates": {
    "latitude": 40.7128,
    "longitude": -74.0060
  },
  "building_height_ft": 35.0,
  "lot_coverage_percent": 40.0,
  "setbacks": {
    "front": 20.0,
    "rear": 25.0,
    "side": 10.0
  },
  "far": 0.4,
  "parking_spaces": 2
}

Code Examples

Python Example

import requests

# Login first (for protected endpoints)
login_response = requests.post('http://localhost:5000/api/auth/login', json={
    'username': 'analyst1',
    'password': 'AnalystGov2024!'
})
token = login_response.json()['session_token']

# Perform compliance analysis
headers = {'Authorization': f'Bearer {token}'}
response = requests.post(
    'http://localhost:5000/api/compliance/analyze',
    headers=headers,
    json={
        'zone_type': 'R-1',
        'coordinates': {'latitude': 40.7128, 'longitude': -74.0060},
        'building_height_ft': 35.0,
        'lot_coverage_percent': 40.0
    }
)

report = response.json()['report']
print(f"Compliance Score: {report['compliance_score']}%")
print(f"Status: {report['overall_status']}")

JavaScript Example

// Login first
const loginResponse = await fetch('/api/auth/login', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({
        username: 'analyst1',
        password: 'AnalystGov2024!'
    })
});
const {session_token} = await loginResponse.json();

// Perform analysis
const response = await fetch('/api/compliance/analyze', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${session_token}`
    },
    body: JSON.stringify({
        zone_type: 'R-1',
        coordinates: {latitude: 40.7128, longitude: -74.0060},
        building_height_ft: 35.0
    })
});

const data = await response.json();
console.log('Compliance Score:', data.report.compliance_score);

Data Sources & Compliance Accuracy

Accuracy Guarantee: Our zoning compliance checker uses only real, verified regulatory data. We never use fake, simulated, or mock data. All compliance calculations are based on official municipal, state, and federal regulations.

Regulation Categories

Building Regulations
  • Height restrictions (max building height)
  • Floor Area Ratio (FAR) requirements
  • Setback requirements (front, rear, side)
  • Lot coverage percentage limits
  • Building density calculations
Parking & Transportation
  • Minimum parking space requirements
  • Accessible parking (ADA compliance)
  • Loading zone requirements
  • Driveway width specifications
  • Transportation impact studies

Data Verification Process

Our compliance data undergoes a rigorous multi-step verification process:

  1. Official Source Collection: We collect regulations directly from official municipal and state government websites and documents.
  2. Legal Review: All regulations are cross-referenced with current zoning ordinances and municipal codes.
  3. Regular Updates: Our database is updated monthly to reflect any changes in local zoning laws.
  4. Accuracy Testing: We perform quarterly audits against official government records.
  5. Professional Review: Licensed urban planners and zoning attorneys review our data quarterly.
Legal Disclaimer: While we strive for 100% accuracy using official government sources, users should always verify compliance with local planning departments before making final development decisions. Zoning regulations can change frequently, and local authorities have the final authority on all compliance matters.
